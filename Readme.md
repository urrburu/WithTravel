# With Travel Project

## 사용중인 기술스택

 spring boot, Java11, spring security, JPA, QueryDsl을 이용한 개발 

## 중점을 두고 개발할 부분
 - JPA에 대해서 공부한 부분 반영해보기

 - DB를 정규화해서 사용하기

 - 스프링, JPA 매끄럽게 사용하기

 - 테스트 케이스 작성하고 사용하기

 - 디버깅을 위한 나만의 방법

 - 추가할 기능 : 여행 경비 계산 및 분담 기능

 - 현재는 여행지를 Set으로 모으고 있는데 List로 바꿔야 할것 같다.
